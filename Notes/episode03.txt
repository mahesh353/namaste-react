    Html or xml like syntax

    JSX makes developers life easy.
    JSX is not html inside javascript.
    JSX is not html.
    JSX is a html like syntax.
    JSX is not valid/Pure javascript. js engine does not understand this code.

    JSX will take care about the xss attacks by sanitizing the data inside {}.
    JSX expression must have only one parent element

Important to know that package manager (NPM) plays a major role in web development. So, we should have a configuration file for npm, that will maintain all packages/dependencies which we need during development & production phase.

1Ô∏è‚É£ package.json - This configuration file is for package manager (npm)
‚úî npm init - this initializer can be used to set up a new or existing npm package
Few important sections of this file -
‚úî Scripts : We can keep few scripts to "starts or build or test" run easily.
"npm run start or npm start" instead of "npx parcel index.html"
üìç npx - it is used to execute the package (one time)
üìç npm - it is used to install/get any package/libraries into project

‚úî Normal dependencies - these are required to run the application on development & production as well.
- npm install react or npm i react
- npm install react-dom or npm i react-dom

‚úî Dev Dependencies - these are required only during development phase.
- npm install --save-dev parcel or npm i -D parcel

2Ô∏è‚É£ Package version details - "^2.9.3" or "~2.9.3" or "2.9.3"
2 - Major version
9 - Minor version
3 - Patches

‚ñ∂ ^ keep this caret symbol if minor and patch wants to get upgraded automatically.
‚ñ∂ ~ keep this tilde symbol if only patch wants to get upgraded automatically.
‚ñ∂ Also, we can keep only the exact version like "2.9.3" if we want get the exact package.
‚ñ∂ Major version cannot be updated automatically, it should be done manually as it will have more impact with new releases.

3Ô∏è‚É£ package-lock.json - It maintains the exact version of the package is installed on development server and also keeps dependent dependencies details.
‚úî Transitive Dependency
React project needs ‚û° parcel (bundler) ‚û° parcel itself needs other dependencies one such dependency is Babel ‚û° Babel also will have its own dependencies ‚û° this is only called "Transitive dependencies"
